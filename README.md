# MLB Stat Track

This repository contains the backend Shiny app code and dependencies for the MLB Stat Track application.

## Live Application

The app is deployed and accessible at: [https://leoy1919.shinyapps.io/mlb_stat_track/](https://leoy1919.shinyapps.io/mlb_stat_track/)

## Overview

This repository includes:
- Shiny app server and UI code
- Required R packages and dependencies
- Data processing scripts
- Supporting functions and utilities

This repository does *not* include:
- Backend flat files (csv) that the app pulls data from
- Scraping and data wrangling scripts that create the flat files that the app pulls from

## Reproducibility

This app is deployed on shinyapps.io. Cloning the repo and running the `app.R` script locally will not recreate the shiny app that is deployed.
